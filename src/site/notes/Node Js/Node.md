Node es un entorno de ejecución de JS fuera del navegador. Este funciona como un [[Node Js/Monohilo\|Monohilo]]

## Motor v8

Entorno de ejecución de JS creado por Google

## **Módulos**

Todos esta basado en [[Node Js/Módulos\|Módulos]]. Muchos módulos vienen por defecto en el paquete de Node.

Además puedes crear tus propios módulos.

## **Orientado a eventos**

Lo clave de node, esta orientado en eventos. Existe un bucle de eventos que se ejecuta constantemente y eventualmente se dispara un evento que podemos escuchar [[Node Js/Event Loop\|Event Loop]], esto nos permite programar de forma reactiva.
